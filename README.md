# AI-Image-Classification
# Train and Bicycle Image Classification

## Overview
This project demonstrates a basic image classification system created with Google Teachable Machine. The model was trained to identify two different categories: **Train** and **Bicycle**. After training, the model was exported in TensorFlow/Keras format and tested in Google Colab using Python to predict the class of new images.

## Categories
- Train
- Bicycle

## Technologies
- Google Teachable Machine
- TensorFlow / Keras
- Python
- Google Colab

## Implementation Steps
1. Created a new image classification project.
2. Defined two categories: Train and Bicycle.
3. Collected and uploaded training images for both categories.
4. Trained the model until it achieved satisfactory results.
5. Exported the trained model in TensorFlow/Keras format.
6. Imported the model files into Google Colab.
7. Ran a Python script to classify test images.
8. Displayed the predicted label along with its confidence score.

## Results
The trained model was able to recognize whether the input image was a **Train** or a **Bicycle**. The prediction result and confidence score were displayed successfully.
